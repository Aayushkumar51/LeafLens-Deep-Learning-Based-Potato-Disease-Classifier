# 🥔 Potato Disease Classification using CNN

An end-to-end deep learning project for classifying potato leaf diseases using a Convolutional Neural Network (CNN).  
This project helps in identifying potato leaf conditions from images and classifies them into:

- **Early Blight**
- **Late Blight**
- **Healthy**

The main goal of this project is to support **smart agriculture** by enabling early disease detection and reducing crop loss.

---

## 📌 Project Overview

Potato crops are highly vulnerable to diseases that can significantly affect yield and quality.  
This project uses **Computer Vision** and **Deep Learning** techniques to automatically detect diseases from potato leaf images.

The model is trained on image data and predicts the disease class with high accuracy.
